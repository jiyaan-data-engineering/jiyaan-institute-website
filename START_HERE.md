## 🐛 Troubleshooting

### Components Not Showing?
→ Check `npm run build` for errors
→ Verify imports in `app/page.tsx`
→ Clear `.next` folder: `rm -rf .next && npm run dev`

### Form Not Working?
→ Check browser console (F12)
→ Verify API endpoint
→ Test with `console.log(form)`

### Mobile View Broken?
→ Test in Chrome DevTools
→ Check responsive classes (sm:, md:, lg:)
→ Test on real device

### Styling Issues?
→ Clear Tailwind cache: `npx tailwindcss init -u`
→ Verify colors in `tailwind.config.ts`
→ Check CSS import order
