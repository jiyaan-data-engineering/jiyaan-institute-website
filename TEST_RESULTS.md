## Support & Troubleshooting

### Common Issues

**Issue**: Form doesn't submit
- **Cause**: Entry IDs are still placeholders
- **Solution**: Update entry IDs in `utils/google-forms.ts`

**Issue**: Demo dates don't update
- **Cause**: Page cache or old browser state
- **Solution**: Clear cache and reload (Ctrl+Shift+Delete)

**Issue**: Success message doesn't appear
- **Cause**: Network request completed but no response status
- **Solution**: Check Network tab in DevTools (expected for Google Forms)
