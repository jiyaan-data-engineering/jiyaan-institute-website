## Mobile vs Desktop Experience

```
MOBILE (< 640px)                    DESKTOP (> 1024px)
─────────────────────────────────────────────────────
1-2 columns per row        ←→      3-4 columns per row
Larger touch targets        ←→      Precise click targets
Stacked layout              ←→      Side-by-side layout
Shorter sections            ←→      Full-width sections
Sticky mobile CTA          ←→      Multiple CTAs visible
Simplified forms           ←→      More detailed forms

MOBILE CONVERSION TIP:
├─ Keep mobile CTA sticky (always visible)
├─ Reduce form fields to 3 (name, email, phone)
├─ Make buttons large (48x48px minimum)
├─ One CTA per section (not overwhelming)
└─ Expected lift: 2-3x better mobile conversion
```
