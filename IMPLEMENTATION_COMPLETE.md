## 🚀 Quick Start Guide

### Step 1: Extract Google Form Entry IDs (5 minutes)

**Option A - Browser Inspect (Easiest)**:
1. Open: https://forms.gle/D8BijjMuaK5ySWHS8
2. Right-click form field → "Inspect"
3. Look for: `<input name="entry.XXXXXXXXX" ...>`
4. Note the entry ID number
5. Repeat for: Email, Name, Phone, Course, Message

**Option B - Network Tab**:
1. Open DevTools (F12) → Network tab
2. Submit the form
3. Find POST request to `/formResponse`
4. Check "Payload" for `entry.XXXXXXXXX` values

### Step 2: Update Configuration

**File**: `utils/google-forms.ts` (lines 14-20)

```typescript
export const FORM_FIELDS = {
  email: "entry.XXXX",      // ← Your Email entry ID
  name: "entry.XXXX",       // ← Your Name entry ID
  phone: "entry.XXXX",      // ← Your Phone entry ID
  course: "entry.XXXX",     // ← Your Course entry ID

}
```

## 💡 Key Design Decisions

### 1. Auto-Generated Fields
**Decision**: Lead ID and Datetimestamp not submitted by form
**Reason**: Google Sheets can auto-generate these via formulas or Apps Script
**Benefit**: Cleaner data flow, automatic tracking

### 2. No Hardcoded Dates
**Decision**: Demo dates calculated dynamically on each page load
**Reason**: Automatic monthly updates without code changes
**Benefit**: Future-proof, zero maintenance

### 3. No Page Redirect
**Decision**: Form uses `fetch` with `no-cors` mode
**Reason**: Better UX, stays on website, shows success message
**Benefit**: Professional feel, increased conversions

### 4. Comprehensive Validation
**Decision**: Client-side validation before submission
**Reason**: Better UX, catches errors early
**Benefit**: Reduced invalid submissions, better user experience

## 🎓 How It Works

### Dynamic Dates Flow
```
Page Load
    ↓
getDemoDates() called
    ↓
Current date analyzed
    ↓
Final Thu-Fri-Sat found
    ↓
Display: "June 2026 - Thursday 25, Friday 26, Saturday 27"
    ↓
Updates automatically every month
```

### Form Submission Flow
```
User fills form
    ↓
Click "Get My Free Counselling Call"
    ↓
Client-side validation
    ↓
Loading spinner shows
    ↓
submitToGoogleForms() called
    ↓
FormData sent to Google Forms endpoint
    ↓
Success message displays (no page redirect)
    ↓
Form resets for next submission
    ↓
Response appears in Google Sheet within seconds
```

## 🔧 Troubleshooting

### Form doesn't submit
1. Verify entry IDs in `utils/google-forms.ts` match your Google Form
2. Check browser console (F12) for errors
3. Verify Google Form is not closed/archived

### Success message doesn't show
1. Check Network tab (F12) - request should reach Google Forms
2. Verify loading state is working
3. Check `components/lead-form.tsx` has correct imports

### Demo dates don't update
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+F5)
3. Check `components/track-record.tsx` imports are correct

### Tests fail
1. Run: `node run-tests.js`
2. Review `TEST_RESULTS.md` for expected behavior
3. Verify `utils/demo-dates.ts` and `utils/google-forms.ts` exist
